CRAG pipeline

  1. Embed & Retrieve — Query embedded via nomic-embed-text; top-k vectors retrieved from ChromaDB.
  2. Grade — Llama 3.1 8B scores each chunk for relevance (strict JSON yes/no gating).
  3. Rewrite & Retry — If no chunks pass, Llama rewrites the query and retries (max 2 loops).
  4. Generate — Command-R7B synthesizes approved chunks into analysis with inline citations.
